Defeat air-gapped systems by exfiltrating data using Apple Find My networkHacker News

A researcher published hzgl-air-bridge, a proof-of-concept tool that exfiltrates data from air-gapped systems by encoding it into Bluetooth Low Energy advertisements that are picked up and relayed by the Apple Find My network. The technique exploits the fact that Apple devices passively relay Find My location packets from unknown devices, turning any nearby Apple hardware into an unwitting data relay. The project demonstrates a covert channel that requires no internet connection on the target machine and no special hardware beyond a Bluetooth-capable device.

Open-Source Tools

sqlite-utils 4.0rc3Simon Willison

Simon Willison released sqlite-utils 4.0rc3, the third release candidate for the major version of his Python library for manipulating SQLite databases from the command line and from code. The most significant new feature in this candidate is support for introspecting and creating compound foreign keys, which introduces a subtle breaking change to the table.foreign_keys API. Willison noted that the changelog since rc2 expanded substantially because he worked through a backlog of issues and pull requests with assistance from Claude Fable 5 and GPT-5.5.

Real Linux. In a browser tab. No install. No server. No DockerHacker News

LinuxOnTab is a project that runs a full Linux environment entirely in a browser tab using WebAssembly, requiring no server-side compute, no Docker container, and no local installation. The environment boots from a compressed filesystem image loaded into the browser and runs processes through a WASM-compiled kernel interface. This approach is notable for enabling reproducible Linux environments in air-gapped or locked-down machines where installing software is not permitted.
