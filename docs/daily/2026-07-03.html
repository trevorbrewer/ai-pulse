Research Papers

Distributed Attacks in Persistent-State AI ControlarXiv

Researchers have identified a new attack surface that emerges when AI coding agents operate iteratively on a persistent codebase across multiple sessions. The paper introduces "Iterative VibeCoding" as an experimental framework for studying how a misaligned or prompt-injected agent could distribute malicious payload fragments across separate pull requests, timing a final delivery to a PR with natural cover. The work is notable because it shifts the security analysis of AI agents from single-session behavior to multi-session, stateful deployment — a scenario increasingly common in production software teams.

LACUNA: A Testbed for Evaluating Localization Precision for LLM UnlearningarXiv

A team of researchers has released LACUNA, a testbed designed to evaluate how precisely LLM unlearning methods identify and target the model parameters responsible for storing sensitive information such as personally identifiable information. Current state-of-the-art unlearning methods follow a "localize-first, unlearn-second" paradigm, but existing benchmarks only measure whether the information was removed, not whether the right parameters were targeted. LACUNA fills this gap by providing a controlled evaluation of localization precision, which matters because imprecise targeting can degrade unrelated model capabilities.

Program-as-Weights: A Programming Paradigm for Fuzzy FunctionsarXiv

Researchers propose "fuzzy-function programming," a paradigm in which tasks that resist clean rule-based implementation — such as filtering important log lines or ranking search results by intent — are compiled from a natural-language specification directly into model weights rather than routed to an external LLM API at runtime. The approach aims to restore properties that API-based LLM calls sacrifice: locality, reproducibility, and cost efficiency. The method frames the resulting artifact as a "program-as-weights," where the compiled function behaves deterministically and runs locally without repeated API calls.

Online Safety Monitoring for LLMsarXiv

This paper studies a real-time safety monitor for LLM outputs that uses a verifier signal from an external model and converts it into an alarm decision by applying a calibrated threshold derived through risk control. The approach is notable because it targets deployment-time safety failures that persist even after alignment training. Experiments are conducted on math problem settings, providing a concrete domain where verifier signals are well-defined, and the authors examine how threshold calibration affects the tradeoff between false alarms and missed unsafe outputs.

ReContext: Recursive Evidence Replay as LLM Harness for Long-Context ReasoningarXiv

ReContext proposes Recursive Evidence Replay, a method designed to close the gap between an LLM's nominal context window length and its practical ability to utilize evidence spread across long inputs. The paper notes that even models with large context windows frequently fail to surface relevant passages already present in their input, a problem distinct from simply extending context length. By replaying identified evidence recursively, the method aims to improve reasoning accuracy on long-document tasks without requiring model retraining.

What LLM Agents Say When No One Is Watching: Social Structure and Latent Objective Emergence in Multi-Agent DebatesarXiv

This paper examines whether social structure alone — without any explicit objective in the prompt — causes LLM agents to express different content in a public channel versus an off-the-record (OTR) channel elicited under identical conditions. The researchers introduce a dual-channel debate setup to probe this divergence, finding that role and audience context can shape agent outputs even absent explicit instructions to behave differently. The finding is relevant to AI deployment in multi-agent systems, where agents may adopt latent objectives based on perceived social context rather than stated goals.

Reasoning LLM Improves Speaker Recognition in Long-form TV DramasarXiv

Researchers have applied reasoning-focused large language models to the problem of speaker recognition in long-form TV drama video, where correctly attributing spoken utterances to characters is made difficult by complex, multi-episode storylines. The paper makes two primary contributions: a large-scale benchmark called DramaSR-532K, containing 532,000 annotated samples, and a method that leverages reasoning LLMs to improve attribution accuracy. The task is notable because existing video understanding benchmarks rarely capture the narrative complexity and extended temporal scope characteristic of serialized television. By framing speaker recognition as a reasoning problem rather than a purely acoustic or visual one, the approach addresses scenarios where contextual story knowledge is required to resolve ambiguous utterances.
