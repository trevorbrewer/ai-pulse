Partition, Prompt, Aggregate: Statistical Self-Consistency in Language ModelsarXiv

This paper examines whether large language model outputs satisfy basic probabilistic identities — specifically the law of total probability — when in-context learning is interpreted as conditional inference. The authors introduce a framework called Partition, Prompt, Aggregate that tests whether prior-weighted conditional distributions estimated by LLMs are internally consistent. Violations of these identities would indicate that the conditional-inference framing of in-context learning is incomplete or misleading, with implications for how practitioners should design prompts and interpret model outputs.

RoboTTT: Context Scaling for Robot PoliciesarXiv

Researchers introduce RoboTTT (Test-Time-Training Robot Policies), a model and training recipe that extends visuomotor context for robot foundation models to 8,000 timesteps — three orders of magnitude beyond current state-of-the-art policies — without increasing inference latency. At this context length, the system enables one-shot in-context adaptation, a capability not previously demonstrated at this scale. Most existing robot foundation models operate with single-step or short-history context, making this a meaningful extension of the architectural design space for robotic control.

MeanFlowNFT: Bringing Forward-Process RL to Average-Velocity GeneratorsarXiv

MeanFlowNFT applies the DiffusionNFT forward-process reinforcement learning framework to MeanFlow generators, which achieve fast few-step sampling by predicting average velocities over time intervals rather than instantaneous velocities. The motivation is to align MeanFlow models with human preferences or task-specific objectives using RL without requiring reverse-process simulation, which is computationally expensive. The paper positions MeanFlow generators as an efficient alternative to standard diffusion models for applications where both speed and reward alignment are priorities.

SciDiagramEdit: Learning to Edit Scientific Diagrams from Paper RevisionsarXiv

SciDiagramEdit addresses the task of automatically editing scientific figures in response to natural-language instructions, treating revisions to published papers as a training signal. The challenge is that scientific figures combine heterogeneous visual elements — labels, panels, and stylistic formatting — making them dense targets for instruction-following models. The dataset is derived from actual manuscript revision histories, giving the benchmark a grounding in real research workflows rather than synthetic editing scenarios.

Online Neural Space Time Memory for Dynamic Novel View SynthesisarXiv

Researchers have proposed an online neural memory system for synthesizing novel views from multi-view streaming video in real time. The work addresses a core tension in dynamic scene reconstruction: maintaining a long-horizon memory of occluded regions while meeting real-time processing constraints. The approach builds on Test-Time Training (TTT) but modifies the standard paradigm, which requires gradient-based memory updates at every frame, to make it tractable for streaming input. By selectively controlling when and how memory updates occur, the method aims to preserve temporal consistency across frames without incurring the full computational cost of per-frame training.
