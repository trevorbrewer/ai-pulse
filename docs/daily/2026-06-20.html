Simon Willison highlighted a Hacker News comment by Sean Lynch arguing that the most meaningful advantage of the Model Context Protocol (MCP) over traditional skills or CLI integrations is its ability to isolate authentication flows outside of an agent's context window. Lynch suggests that keeping auth out of the context window reduces complexity and potential exposure, and that even a stripped-down version of MCP functioning solely as an auth gateway for an API would represent a net improvement. The observation reframes MCP's value proposition away from broad tool-calling capabilities and toward a narrower, security-oriented use case.
