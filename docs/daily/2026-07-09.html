Accurate, Interdisciplinary and Transparent Structure-property Understanding with Deep Native Structural ReasoningarXiv

Researchers propose a framework for modeling structure-property relationships across biology, chemistry, and materials science using what they call deep native structural reasoning. The approach aims to make model reasoning mechanistically interpretable by grounding inference in physical and chemical constraints such as stereochemistry, bonding, symmetry, and energetics. This is notable because most deep learning approaches to molecular and materials property prediction treat the model as a black box, whereas this work targets transparent, principle-grounded explanations.

Co-LMLM: Continuous-Query Limited Memory Language ModelsarXiv

The paper introduces Co-LMLM, an extension of limited memory language models (LMLMs), which externalize factual knowledge to a knowledge base rather than encoding it in model weights during pretraining. The proposed continuous-query mechanism allows the model to retrieve from the knowledge base dynamically throughout generation, addressing limitations of the original LMLM paradigm. This design enables finer-grained knowledge control compared to conventional LLMs, a property useful for applications requiring verifiable or updatable factual content.

The Key to Going Linear: Analysis-Driven Transformer LinearizationarXiv

This paper investigates how to replace the quadratic causal self-attention mechanism in transformers with linear alternatives without degrading model quality. Working under a strict frozen-backbone regime — where model weights are not retrained — the authors isolate the role of state update design and show that standard softmax attention relies on key-dependent, rank-1 orthogonal projections. The findings provide a principled basis for selecting which linearization components preserve performance, addressing a gap in the existing post-hoc linearization literature.

From Noisy Traces to Root Causes: Structural Trajectory Analysis and Causal Extraction for Agent OptimizationarXiv

The paper addresses the challenge of using execution traces from long-horizon AI agents to diagnose failures and improve agent policies via LLM-based optimizers. Raw trace collections are often redundant and heterogeneous, making direct use for optimization inefficient and prone to overfitting. The authors propose structural trajectory analysis and causal extraction methods to surface root causes from noisy traces, aiming to make reflection-based agent optimization more reliable and data-efficient.

Breaking Database Lock-in: Agentic Regeneration of High Performance Storage Readers for Database BypassarXiv

Researchers present Jailbreak, a system that bypasses database engines entirely by reading storage files directly, circumventing the bottleneck imposed by drivers like JDBC and ODBC when handling bulk columnar analytics workloads. The approach uses agentic code generation to regenerate high-performance storage readers, enabling analytical queries to skip query execution layers not designed for such access patterns. This is notable because database driver lock-in is a longstanding constraint for data-intensive pipelines that rely on external storage systems.

Institutional Red-Teaming: Deployment Rules, Not Just Models, Causally Shape Multi-Agent AI SafetyarXiv

The paper introduces institutional red-teaming, a methodology that tests deployment rules in multi-agent AI systems by holding agents, objectives, and task state fixed while varying a single rule at a time. The authors instantiate this approach with IABench-CA, a benchmark covering 228 contexts, five canonical rules, and seven model providers, to evaluate how collective agent behavior changes causally in response to rule changes. The work argues that safety analysis focused solely on model capabilities misses a significant source of risk: the institutional and policy layer governing how agents are deployed.

Selective Timestep Weighting and Advantage-Based Replay for Sample-Efficient Diffusion RLHFarXiv

This paper proposes two techniques — selective timestep weighting and advantage-based replay — to reduce the feedback inefficiency of applying reinforcement learning from human feedback (RLHF) to diffusion models. Existing diffusion RLHF methods typically require large numbers of reward model evaluations for each update, limiting practical use in settings where labeled feedback is scarce. The selective weighting scheme focuses training signal on the most informative diffusion timesteps, while advantage-based replay prioritizes high-quality generated samples for reuse, together reducing overall feedback requirements.

Agon: Competitive Cross-Model RL with Implicit Rival Grading of ReasoningarXiv

Agon introduces a competitive reinforcement learning setup in which two language models grade each other's reasoning traces rather than only evaluating final answers. Standard verifiable-reward approaches such as GRPO grade only the output, which the authors argue incentivizes longer responses on hard problems rather than better reasoning. In the Agon framework, both models attempt the same problem and each serves as the other's implicit grader, creating a signal for trace quality without requiring human-labeled examples of good reasoning.

ECGLight: Compute-Light Framework For Paper ECG Digitization and Myocardial Infarction ScreeningarXiv

Researchers have proposed ECGLight, a lightweight AI framework designed to digitize paper ECG printouts and screen for myocardial infarction in resource-constrained clinical settings. The work addresses a practical gap: many remote clinics still rely on physical ECG printouts due to limited internet connectivity and low computational capacity, placing them outside the reach of existing AI diagnostic tools. ECGLight is specifically architected to run under these constraints, enabling digitization and analysis without requiring cloud infrastructure or high-end hardware. The framework targets cardiovascular disease screening, with myocardial infarction — a time-sensitive condition — as its primary clinical use case.
