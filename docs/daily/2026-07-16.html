xAI open-sourced its Grok CLI coding agent, but the release was immediately overshadowed by reports that running the tool in a directory caused it to upload the entire directory's contents to xAI's Google Cloud storage buckets. At least one user reported that this included sensitive files such as SSH keys, a password manager database, and personal media. Mermaid to Unicode box art (grok-mermaid)Simon Willison

While examining the open-sourced Grok CLI codebase, Simon Willison discovered a Rust-based terminal renderer for Mermaid diagrams called grok-mermaid, located at xai-grok-markdown/src/mermaid.rs. He compiled it to WebAssembly and ran it in a browser to render Mermaid diagrams as Unicode box art, using Claude Code for web (Fable 5) to assist with the conversion. The experiment demonstrates a secondary use of components from the Grok CLI release independent of the privacy controversy surrounding the main tool.

Research Papers

Deep Interaction: An Efficient Human-AI Interaction Method for Large Reasoning ModelsarXiv

Researchers have proposed Deep Interaction, a method designed to make human correction of large reasoning models more efficient during Chain-of-Thought (CoT) inference. Current approaches to fixing model errors are costly: either the model regenerates a full response that may repeat the same mistake, or users must manually identify the faulty reasoning step across multiple follow-up turns. Deep Interaction addresses this by enabling more targeted, mid-reasoning intervention, allowing users to correct errors at the specific step where they occur rather than restarting the entire generation. The work is notable because it directly targets the interaction overhead introduced by long-form CoT reasoning, which has become common in frontier models.
