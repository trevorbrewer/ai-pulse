ELSA3D: Elastic Semantic Anchoring for Unified 3D Understanding and GenerationarXiv

Researchers have proposed ELSA3D, a unified 3D foundation model architecture that aims to handle both 3D asset generation and language-based reasoning within a single backbone. The work identifies a core weakness in existing approaches: concatenating text and 3D tokens into a flat sequence for self-attention collapses coarse structural cues and fine geometric details into one undifferentiated representation. ELSA3D addresses this by introducing elastic semantic anchoring, a mechanism designed to maintain explicit structure in text-3D interactions rather than leaving them implicit.

Bridging Physical Reasoning and Task Generalization via Visual Action Outcome Reasoning AlignmentarXiv

The paper introduces VAORA (Visual Action Outcome Reasoning Alignment), a reward-design framework targeting two identified failure modes in vision-language models operating in interactive physical settings: chain-of-thought reasoning that contradicts physical reality, and misalignment between a model's stated reasoning and its actual actions. The work focuses specifically on generalization under unseen tasks and environments, a known challenge for current VLMs. By aligning visual action outcome reasoning, VAORA attempts to ground model inference in physically consistent predictions rather than hallucinated rationales.

GraphBU: MILP Instance Generation with Graph-Native Block UnitsarXiv

GraphBU proposes a new method for generating mixed-integer linear programming (MILP) instances intended to support solver development when real-world instances are unavailable due to privacy or proprietary constraints. Existing generators typically derive their structure from formulation templates, summary statistics, local graph edits, or decomposed blocks, but these approaches can fail to preserve the structural properties that solvers and learned policies rely on. GraphBU uses graph-native block units as its generation primitive, aiming to better capture the structural characteristics of practical MILP instances.

Research Papers: Language Models & NLP

RSF-GLLM: Bridging the Semantic Gap in Multi-Hop Knowledge Graph QA via Recurrent Soft-Flow and Decoupled LLM GenerationarXiv

RSF-GLLM is a framework for multi-hop question answering over knowledge graphs that decouples differentiable graph reasoning from answer generation, addressing a core limitation of traditional retrieve-then-read pipelines. The key problem those pipelines face is that breaking differentiability prevents the retriever from learning to bridge semantic gaps, particularly at intermediate nodes that lack lexical overlap with the query. The proposed Recurrent Soft-Flow mechanism is designed to maintain end-to-end gradient flow through the graph traversal process.

DepthWeave-KV: Token-Adaptive Cross-Layer Residual Factorization for Long-Context KV Cache CompressionarXiv

DepthWeave-KV introduces a compression method for key-value caches in long-context language model inference, targeting the memory bandwidth and capacity costs that increasingly constrain deployment. Unlike existing methods that apply uniform compression budgets across all layers or tokens, DepthWeave-KV uses token-adaptive factorization that distributes cache budgets differently depending on whether a given layer or token requires preservation of lexical cues or semantic states. The cross-layer residual factorization approach is intended to reduce degradation in retrieval-heavy tasks that uniform methods are known to harm.

FreqDepthKV: Frequency-Guided Depth Sharing for Robust KV Cache Compression in Long-Context LLM InferencearXiv

Researchers have proposed FreqDepthKV, an inference-time method for compressing the key-value (KV) caches that consume significant memory and bandwidth during long-context large language model inference. The approach factorizes KV states from adjacent layers into shared low-frequency depth components, which capture broad contextual patterns, and sparse high-frequency residuals that preserve layer-specific detail needed for retrieval and multi-step reasoning. This decomposition is notable because aggressive KV cache compression has previously risked discarding the fine-grained, layer-specific information critical to complex reasoning tasks. The method operates at inference time, requiring no retraining, which lowers the barrier to adoption across existing deployed models.
